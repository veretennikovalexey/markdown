## Комментарии

| Вариант | Способ      | Синтаксис               |
|--------:|-------------|-------------------------|
|       1 | **html**    | `<!-- комментарий -->`  |
|       2 | **comment** | `[comment]: # "текст"`  |
|       3 | **//**      | `[//]: # (комментарий)` |
|       4 | **^_^**     | `[^_^]: <> 'комментарий'` |
