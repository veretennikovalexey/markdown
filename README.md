## Комментарии

\<!--  этот текст виден не будет  
\-->

\[comment]: # "этот текст виден не будет"  

\[//]: # (Еще один вариант)


| Вариант | Способ      | Синтаксис               |
|--------:|-------------|-------------------------|
|       1 | **html**    | `<!-- комментарий -->`  |
|       2 | **comment** | `[comment]: # "текст"`  |
|       3 | **//**      | `[//]: # (комментарий)` |
